Sky Candy Rigging Quiz

Project Description: Built using jQuery, this web app is designed to test the users knowledge of rigging aerial apparatus at Sky Candy, an aerial studio in Austin, TX. It will be taken by Sky Candy employees following their mandatory rigging training. I collaborated with the CEO and owner on the questions and I used Sky Candy's website to inform the design.
